To include footer to your webpage

[1]. Add the style sheet link in below in the head tag

    <!-- footer css link -->
    <link rel="stylesheet" href="./footer/footer.css">
    <!-- footer css link -->

[2]. Add the js script tag just before the closing body tag </body>

    <!-- footer script link -->
    <script src="./footer/footer.js"></script>
    <!-- footer script link -->

[3]. Add footer tag after the last components in the body
        ++ footer must include as the last component ++

    <footer></footer>
